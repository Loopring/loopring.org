Contribution

How to contribute

We use Slack to communicate with each other. Feel free join in our slack channels:

  • Loopring Announcement (read-only)
  • Loopring Community (English)
  • Loopring Contributors

You may want to talk to Daniel Wang (daniel@loopring.org), Loopring CEO and major developer, about your idea and the way you can help. This is not a prerequisite, however, you can create pull requests at anytime.

Our Git Branching Model

For git repositories that have releases (this one not included), we will adopt "A Successful Git Branching Model" starting from Nov. 1st, 2017.
