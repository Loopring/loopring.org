External Documentations

Protocol

  • Loopring whitepaper
  • https://loopring.github.io/protocol: the Loopring Protocol implementation on Ethereum.
  • https://loopring.github.io/neo-protocol: the Loopring Protocol implementation on NEO.

Frontend

  • https://loopring.github.io/loopring.js: the Loopring Javascript library built for accessing the Loopring Protocol.
  • https://loopring.github.io/loopr: web Loopr wallex.
  • https://loopring.github.io/circulr: web-based DEX frontend circulr.
  • https://loopring.github.io/h5dex: embedable DEX solution h5dex.

Backend

  • https://loopring.github.io/relay-cluster: distributed relay cluster.
  • https://loopring.github.io/miner: miner component, which is part of the relay cluster.
  • https://loopring.github.io/extractor: extractor component, which is part of the relay cluster.
  • https://loopring.github.io/relay-lib: reusable components for the relay cluster.

Missilenious

  • https://loopring.github.io/incentives: smart-contract based incentivization programs.
  • https://loopring.github.io/docs: other missilenious documentations.
